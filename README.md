# Rick and Morty Backend (ExpressJs)

## Description

This is an ExpressJs App build-in with ReactJs and Graphql. On this api, you can get Rick and Morty...
* Characters
* Episodes
* Locations

**Nota:** For now, just there Read function from CRUD functionability.

## Instalation

1. Run `npm i`

## Running the app

```bash
# build
$ npm run build

# development mode
$ npm run start

# run tests
$ npm run test

# run eject
$ npm run eject
```
